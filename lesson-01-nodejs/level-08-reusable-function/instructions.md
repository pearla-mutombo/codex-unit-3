# Reusable Function

## Objective

Refactor code into a reusable function.

## Benefits

Practice to get better with functions and refactoring.

## Complete these tasks

1. Write the code from scratch. Do not copy/paste.
2. In `script.js`, create an array with 10 values.
3. Add a `for` loop with a starting index and an ending condition.
4. Use the loop to traverse the array.
5. Display each value in the array, but number them from 1 to 10.
6. Run the script. It should run without errors.
7. Create a function called `showItems`.
8. Implement the function to show the items of an array.
9. Create another array.
10. Let `showItems` display that new array.
11. Run the script.

## More Information

- Reusable functions should take in parameters or return values that may change.

## Usage Tips

- Try creating a function without parameters without parameters or a return value.
- Add parameters or a return value as needed.

## Hints

- `showItems` should work for any array.
