# React Variables

## Objective

Use local variables in a React page.

## Benefits

In React, local variables can be directly inserted into HTML tags.

## Complete these tasks

1. In your system's file explorer or file finder, copy your `react-template` folder into this level folder.
2. In this level folder, rename `react-template` to `react-variables`.
3. In the terminal, navigate to the `react-variables` folder.
4. Start the Vite server with `npm run dev`.
5. Open a web browser to the URL provided by Vite.
6. In `App.jsx`, let the `App` function return a `main` tag.
7. In the `main` tag, add an `h1` tag, and 2 `p` tags.
8. Let the `h1` title be `React Variables`.
9. For now, let the `p` tags contain empty curly braces `{}`.
10. View the page in the browser.
11. Above the `return` statement, create 2 string varables - `message1` and `message2`.
12. Let `message1` explain how to create React variables. Use your own words. Don't copy/paste.
13. Let `message2` explain how to use React variables. Use your own words. Don't copy/paste.
14. Insert the message variables into the empty curly braces. Example: `<p>{message1}</p>`
15. View the page in the browser.
16. In `App.css` or `index.css`, add at least 2 styles to improve the website's layout and appearance.
17. View the page in the browser.

## More Information

- In React, curly braces `{}` embedded in HTML can contain JavaScript code.
- In this exercise, the curly braces contain JavaScript variables.

## Usage Tips

- In React, and JavaScript in general, local variables can be created within functions.
- In React, variables can be inserted into HTML with curly braces. Example: `<h1>{myTitle}</h1>`

## Hints

- In React, and JavaScript in general, global variables can also be used.
