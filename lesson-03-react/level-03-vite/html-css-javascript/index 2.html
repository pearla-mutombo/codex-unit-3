Vite Scaffold is where all of your starting filed are automatically created from a template that is imported from a specified src script.js and regular scaffolding is were you create your own template page with your own starting files. The Difference:my manual scaffold is just a set of basic files that i open like a regular document on my computer. The Vite scaffold comes with its own-website launcher. It automatically refreshes the page the second I save changes, cleans up my code behind the scenes, and uses a JavaScript file to build and insert the website content on the fly. basicslly a manual scaffold is a static file you open straight from your hard drive. a Vite scaffold uses a build-in live server that auto-updates your screen instanly when you save code. instead of writing everything in HTML, Vite uses a JavaScript file to automatically inject the text and layout into the page.
