# Vite

## Objective

Scaffold a new project with Vite.

## Benefits

Vite use used to scaffold a variety of projects.

## Complete these tasks

1. In a VS Code terminal, navigate to this level folder.
2. Use `npm` to create a `vite` project.
3. Name the project `html-css-javascript`.
4. Let it be a `Vanilla` and `JavaScript` project.
5. Let it automatically install dependencies.
6. Open a web browser to the URL provided by Vite.
7. View the code in the `html-css-javascript` folder.
8. Observe how `index.html` and `main.js` renders the page.
9. Observe how the code in `main.js` looks like HTML code.
10. In `index.html`, add a `p` tag that renders a message.
11. Let the message explain the difference between your scaffold and the Vite scaffold.
12. View the message in the browser.

## More Information

- Vite is a tool that makes starting, developing, and building web apps much faster.
- Vite provides boilerplate and scaffolding for common project types.
- Vite provides its own server for previewing websites.

## Usage Tips

- Use `npm create` and a project type to scaffold a project. Example: `npm create some-type`
- To stop the Vite server, click on the terminal and press `CTRL + C`, or close the terminal.

## Hints

- Vite may not use proper semantic tags in the body tag.
- To manually start a Vite server, in the terminal, navigate to the project folder, then type npm run dev.
