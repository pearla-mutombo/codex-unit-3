# Scaffold

## Objective

Scaffold a basic web project.

## Benefits

Scaffolding a web project can simplify the development of a similar web project.

## Complete these tasks

1. In the VS Code file list, select this level folder.
2. Create a folder called `html-css-javascript`.
3. In `html-css-javascript`, create the file `index.html`.
4. In `index.html`, add boilerplate tags and an `h1` tag with `HTML, CSS, JavaScript Template`.
5. Create `style.css`.
6. Let `style.css` make a style change to the `h1` tag.
7. Use the `link` tag to connect `style.css` to `index.html`.
8. Create `script.js`.
9. In `script.js`, use console log to display a message.
10. Let the message explain scaffolding.
11. In the `head` tag of `index.html`, use the `script` tag and `defer` to connect `script.js`.
12. Preview `index.html`.

## More Information

- Scaffolding is creating a folder with the initial files and folders commonly needed to begin developing.
- With scaffolding, a copy of the folder can be used as the starting template of a new project instead of starting with an empty folder.

## Usage Tips

- Use `src` to specify JavaScript files.
- Use `href` to specify CSS files.

## Hints

- `!` can be used to quickly add boilerplate HTML tags.
