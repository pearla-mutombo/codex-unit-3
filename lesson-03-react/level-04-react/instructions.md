# React

## Objective

Scaffold a React project with Vite.

## Benefits

Using Vite as a tool to scaffold a React project can speed up development of web projects.

## Complete these tasks

1. In a VS Code terminal, navigate to this level folder.
2. Use `npm` to create a `vite` project.
3. Name the project `react`.
4. Let it be a `React` and `JavaScript` project.
5. Let it automatically install dependencies.
6. Open a web browser to the URL provided by Vite.
7. View the code in the `react` folder.
8. Observe how `index.html` and `main.jsx` renders the page.
9. Observe how the code in `App.jsx` looks like HTML code.
10. In `index.html`, add a `p` tag that renders a message.
11. Let the message explain the difference between your scaffold and the Vite React scaffold.
12. View the message in the browser.

## More Information

- React is used to build websites with reusable components.
- Components are like modules or snippets of HTML code.
- React combines JavaScript and HTML-like syntax to provide a unified development environment.
- React give HTML-like code direct access to JavaScript variables.

## Usage Tips

- To manually start a Vite server, in the terminal, navigate to the project folder, then type `npm run dev`.

## Hints

- The JSX syntax is JavaScript, but it closely resembles HTML.
- To stop the Vite server, click on the terminal and press `CTRL + C`, or close the terminal.
